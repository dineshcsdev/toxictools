Frequently Asked Questions

ToxicTools leverages OpenRouter API to access various models. Currently, it uses 'firellava/firellava-13b' for image-based roasts and 'deepseek/deepseek-r1' for all text-based interactions.

We do not store any user input or generated responses. All interactions are processed in real-time and are not logged on our servers.

We welcome contributions! Please check our CONTRIBUTING.md guide on GitHub for details on how to get started, report bugs, or suggest features.
